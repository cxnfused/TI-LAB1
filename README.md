# README — Шифрование текста (Плейфейр и Виженер с прогрессивным ключом)

## 📋 О проекте
Графическое приложение (tkinter) для шифрования и дешифрования текстовых файлов и ввода с клавиатуры.  
Поддерживаемые алгоритмы:
- **Playfair** — подстановочный шифр (английский алфавит, 25 букв, J→I)
- **Виженер с прогрессивным ключом** — подстановочный шифр (русский алфавит, 33 буквы, включая Ё)

Программа автоматически игнорирует все символы, не являющиеся буквами соответствующего алфавита, а также поддерживает загрузку и сохранение `.txt` файлов.

## 🎮 Инструкция по использованию

### Шаг 1. Выберите алгоритм
Если хотите... | Выберите | Ключ
--- | --- | ---
Шифровать заменой по таблице | **Плейфейр (английский)** | Слово (только A–Z, J→I)
Шифровать прогрессивным ключом | **Виженер (русский)** | Слово (только русские буквы, включая Ё)

### Шаг 2. Введите ключ
- **Playfair**: `KEY` (можно использовать строчные/прописные буквы, цифры и другие символы будут отфильтрованы).
- **Виженер**: `КЛЮЧ` (только русские буквы; остальные будут проигнорированы).

**Важно:**  
- Длина ключа не должна превышать 100 символов.  
- Ключ не может быть пустым.

### Шаг 3. Загрузите текст
Три способа:
1. Напечатать вручную в поле «Исходный текст».
2. Нажать **«Файл → Открыть…»** и выбрать `.txt` файл.
3. Использовать любой текст, который уже есть в поле.

Все не‑буквенные символы будут удалены автоматически; при необходимости программа сообщит, сколько символов отфильтровано.

### Шаг 4. Шифруйте / Дешифруйте
Кнопка | Что делает
--- | ---
**Зашифровать** | Превращает исходный текст в шифровку.
**Расшифровать** | Восстанавливает исходный текст (при правильном ключе).

### Шаг 5. Покажите процесс
Нажмите **«Показать процесс»** — откроется отдельное окно с детальной визуализацией:
- Для Playfair: таблица 5×5 и пошаговый разбор всех пар.
- Для Виженера: пошаговая таблица (символ, базовый ключ, блок, фактический ключ, индекс, результат) и большая таблица 33×33 для наглядности.

### Шаг 6. Сохраните результат
Нажмите **«Файл → Сохранить результат…»** — браузер (или файловая система) предложит сохранить `.txt` файл с результатом.

## 🔐 Алгоритмы

### Playfair (английский)
- Ключ очищается от повторяющихся букв (J заменяется на I), затем таблица 5×5 заполняется оставшимися буквами алфавита.
- Текст разбивается на пары (нечётный символ дополняется X).
- **Правила шифрования пары**:
  - одна строка → сдвиг вправо
  - один столбец → сдвиг вниз
  - иначе → прямоугольник (обмен столбцами)
- Дешифрование выполняется аналогично, но сдвиг влево/вверх.

### Виженер с прогрессивным ключом (русский)
- Алфавит: `АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ` (33 буквы).
- Прогрессивный ключ формируется так:
  - базовый ключ повторяется циклически;
  - каждый следующий блок сдвигается на единицу (например, `АБВГ` → `БВГД` → `ВГДЕ` …).
- **Шифрование**: `результат = (индекс_буквы + (индекс_базового_ключа + номер_блока)) mod 33`
- **Дешифрование**: `результат = (индекс_буквы - (индекс_базового_ключа + номер_блока) + 33) mod 33`

## 📂 Работа с файлами
- **Открыть** – загружает содержимое `.txt` файла в поле ввода.
- **Сохранить результат** – сохраняет содержимое поля результата в `.txt` файл.
- При попытке открыть бинарный файл выводится сообщение об ошибке.

## ⚙️ Требования
- Python 3.6 или выше.
- Стандартная библиотека (tkinter, os, filedialog, messagebox) – дополнительные установки не требуются.

## 🚀 Запуск
```bash
python crypto_app.py
```

## 📱 Ограничения
- Максимальная длина ключа – 100 символов.
- Для Playfair ключ должен содержать хотя бы одну английскую букву (A–Z).
- Для Виженера ключ должен содержать хотя бы одну русскую букву (включая Ё).
- Текст без подходящих букв не обрабатывается (выводится предупреждение).
